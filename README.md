## üìù How to Use

### Quick Start

1. **Install the Extension**
   - Install from [Chrome Web Store](https://chromewebstore.google.com/detail/lmhbkobhomjjikgkdfpjkcmpafikopbe)

2. **Connect to Notion**
   - Click the extension icon
   - Click "Connect to Notion"
   - Authorize the extension to access your workspace

3. **Select a Database**
   - Choose your target Notion database from the dropdown
   - The extension will automatically add necessary fields

4. **Start Saving**
   - **Single Save**: Click the extension icon on any Claude chat page and hit "Save to Notion"
   - **Batch Save**: Right-click the extension ‚Üí Options ‚Üí Configure sync settings ‚Üí Click "Sync Now"

### Sync Modes

Choose how to handle existing conversations:

- **Skip Existing**: Skip conversations that already exist (recommended to avoid duplicates)
- **Overwrite**: Delete existing pages and recreate them with fresh content
- **Always Create New**: Create new pages even if duplicates exist

### Configuration Options

- **Sync Scope**: Choose between latest 30 conversations or all conversations
- **Enhanced Formatting**: Use toggle lists for cleaner display
- **Skip Synced Records**: Avoid re-syncing conversations in the current session
